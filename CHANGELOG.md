## [0.1.0] - 2026-07-07

### Added

- 初回MVPリリース
- 請求書の作成フォーム（発行者・請求先・明細・備考・振込先）
- インボイス制度（適格請求書）対応:
  - 登録番号（T+13桁）の記載欄
  - 税率ごと（10% / 軽減8%）の区分記載と消費税額の自動計算（税率ごとに端数切り捨て）
  - 軽減税率対象品目の「※」表示
- 源泉徴収税額の自動計算（100万円以下 10.21% / 超過分 20.42% の2段階に対応）
- ブラウザの印刷機能によるA4・PDF出力（印刷用CSS）
- localStorage への自動保存（登録不要・サーバー送信なし）
- 請求書データの JSON エクスポート / インポート
- 開発用の依存ゼロ静的サーバー（`npm run dev`）
- 計算ロジックのユニットテスト（`npm test`）
